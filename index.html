Semantic Tags

Tag Usage
<article> Denotes an article.
<aside> Denotes content displayed in a sidebar of the webpage.
<figcaption> A description of the figure is placed inside <figure>.
<figure> A tag reserved for figures (diagrams, charts).
<footer> Specifies the webpage footer. Could also be used for objects inside the webpage.
<header> Specifies the webpage header. Could also be used for objects inside the webpage.
<main> Marks the main content of the webpage.
<nav> Navigation links for the user in a webpage.
<section> Specifies a particular section in the webpage.
